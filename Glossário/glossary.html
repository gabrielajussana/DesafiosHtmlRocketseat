Glossário de Tags HTML

<!DOCTYPE html >
Uma declaração que indica ao navegador qual a versão
utilizada no documento HTML.
<html>
Representa a raiz de um HTML ou XHTML documento.
Todos os outros elementos devem ser descendentes desse elemento.
<head>
Providencia informações gerais (metadados) sobre o documento,
incluindo seu título e links para scripts e folhas de estilos.
<body>
Define o corpo do documento html.
<p>
Representa um parágrafo.
